# 🤖 SupportPilot AI

> AI-powered customer support CRM built with Flask, Python, JavaScript, and Google Gemini.

SupportPilot AI is a modern customer support dashboard that combines CRM functionality with AI-assisted ticket analysis and response generation. It helps support teams manage customers, track tickets, and generate professional replies using Google Gemini.

---

## ✨ Features

* 📊 Executive Dashboard
* 👥 Customer Management
* 🎫 Ticket Management
* 🤖 AI Copilot (Gemini Powered)
* 📝 AI Ticket Summary
* 😊 Sentiment Analysis
* 🏷️ Smart Tag Suggestions
* 🎯 Confidence Score
* 🏢 Department Recommendation
* 🚨 Priority Prediction
* ⏱️ Resolution Time Estimate
* ❤️ Customer Health Score
* 📋 One-click AI Reply Generation

## 🚀 Installation

```bash
git clone https://github.com/YOUR_USERNAME/supportpilot-ai.git

cd supportpilot-ai

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

python app.py
```

Open:

```
http://127.0.0.1:5000
```
